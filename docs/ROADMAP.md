# Roadmap

## Milestone 1: Foundation
- workspace bootstrap
- Electron menubar shell
- typed preload bridge
- current session persistence and recents
- recursive intake and asset indexing

## Milestone 2: Core Compare
- single viewer
- 2-up, 3-up, 4-up layouts
- background, zoom, fit mode, metadata, sync controls
- grid browser for folder and large-set flows

## Milestone 3: Diff
- pixelmatch image diff
- text/code diff with `@pierre/diffs`
- normalization messaging and unsupported-state handling

## Milestone 4: QA And Packaging
- unit and component coverage
- packaged Electron smoke automation
- Apple Silicon `.app` and `.zip` output
- doc completion for architecture, data model, UX, and testing
